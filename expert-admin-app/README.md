# 専門家管理アプリ

「きょうのできた」の専門家相談機能を管理するアプリケーション

## 機能

- 専門家の登録・編集・削除
- 予約管理
- お知らせ送信（親アプリのホーム画面に表示）
- 統計ダッシュボード

## セットアップ

1. 依存関係のインストール
```bash
npm install
```

2. 環境変数の設定
`.env`ファイルを作成し、Supabaseの接続情報を設定

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

3. 開発サーバーの起動
```bash
npm run dev
```

## 使い方

### 専門家の追加

1. 「専門家管理」タブを開く
2. 「専門家を追加」ボタンをクリック
3. 必要事項を入力して保存
4. 追加された専門家は自動的に親アプリに反映されます

### お知らせの送信

1. 「お知らせ」タブを開く
2. タイトルと本文を入力
3. 「お知らせを送信」ボタンをクリック
4. すべての親アプリユーザーに通知が届きます

## データベース連携

このアプリは`experts`テーブルと直接連携しています。
専門家を追加・編集すると、親アプリの専門家一覧に即座に反映されます。
