## 🐛 エラーが出た場合

### エラー: "relation does not exist"
→ データベーステーブルが作成されていません
→ `/shared/minimal_database_setup.sql` を再実行

### エラー: "permission denied"
→ RLS設定を確認
→ Supabase Settings → Database → Row Level Security

### データが表示されない
→ ブラウザをリロード（F5）
→ コンソール（F12）でエラーを確認

### 画像が表示されない
→ Base64データが正しく保存されているか確認
→ 画像ファイルサイズが5MB以下か確認
